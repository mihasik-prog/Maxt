## 📱 О проекте

Max Tracker позволяет отслеживать, к каким IP-адресам обращается любое установленное приложение на вашем Android-устройстве. Он создаёт локальный VPN-туннель (без root-прав) и показывает все исходящие соединения выбранной программы.

Проект родился из желания проверить мифы о «шпионаже» мессенджеров и магазинов приложений. С его помощью я лично провёл расследование и выяснил, куда на самом деле утекают данные RuStore, TikTok и других приложений.

## ✨ Возможности

- 🎯 Отслеживание исходящих IP-адресов любого выбранного приложения.
- 🔒 Работа без root-прав через локальный VPN.
- 📋 Простой и понятный интерфейс.
- ℹ️ Встроенные разделы с политикой конфиденциальности, лицензией и информацией о создателях.

## 📥 Установка

1. Скачайте последнюю версию APK из раздела [Releases](https://github.com/mihasik-prog/Maxt/releases).
2. Установите APK на устройство (Android 11+).
3. При первом запуске разрешите создание VPN-подключения.

## 🧪 Как использовать

1. Введите имя пакета интересующего приложения (например, `ru.vk.store` для RuStore).
2. Нажмите «Start Monitoring» и подтвердите VPN-запрос.
3. Откройте целевое приложение и активно пользуйтесь им.
4. Наблюдайте за списком IP-адресов, к которым обращается программа.
